# 🤖 Joca do MARCOS - Instalação Rápida

## ⚡ 3 Passos Simples

### 1️⃣ Compilar o APK

```bash
cd /home/ubuntu/joca-agent-app
eas build --platform android --local
```

Ou use EAS Build na nuvem (mais rápido):
```bash
eas build --platform android
```

### 2️⃣ Transferir para o Celular

- Baixe o arquivo `.apk` gerado
- Envie para seu celular via email, WhatsApp ou USB

### 3️⃣ Instalar e Configurar

1. Abra o arquivo `.apk` no celular
2. Toque em **Instalar**
3. Siga o **Checklist de Permissões** que aparecerá
4. Conceda todas as permissões solicitadas
5. Pronto! O Joca está pronto para usar 🎉
